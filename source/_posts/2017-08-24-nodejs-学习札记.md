## 服务器配置 CSP 安全策略
```js
if (config.CSP_HEADER) {
  res.set({
    'Content-Security-Policy': config.CSP_HEADER,
    'X-Content-Security-Policy': config.CSP_HEADER,
    'X-WebKit-CSP': config.CSP_HEADER
  });
 }
```
