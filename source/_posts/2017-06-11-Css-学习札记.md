### 一、选择器
- `*{}` 通配符选中所有 **元素**
- h1,h2...{} 元素选择器
- .className{} 类选择器
- #idName{} id 选择器，只能有一个，不能结合使用
- 后代选择器 空格
- 第一子代选择器 >
- 相邻兄弟选择器
*  h1+p{} // 如果需要选择紧接在另一个元素后的元素，而且二者有相同的父元素，可以使用相邻兄弟选择器
```css
例1：如果要增加紧接在 h1 元素后出现的段落的上边距，可以这样写：
h1 + p {margin-top:50px;}
这个选择器读作：“选择紧接在 h1 元素后出现的段落，h1 和 p 元素拥有共同的父元素”

例2：
<div>
  <ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
  </ul>
  <ol>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
  </ol>
</div>
在上面的片段中，div 元素中包含两个列表：一个无序列表，一个有序列表，每个列表都包含三个列表项。这两个列表是相邻兄弟，列表项本身也是相邻兄弟。不过，第一个列表中的列表项与第二个列表中的列表项不是相邻兄弟，因为这两组列表项不属于同一父元素（最多只能算堂兄弟）。
请记住，用一个结合符只能选择两个相邻兄弟中的第二个元素。请看下面的选择器：
li + li {font-weight:bold;}
上面这个选择器只会把列表中的第二个和第三个列表项变为粗体。第一个列表项不受影响。
// ref: http://www.w3school.com.cn/css/css_selector_adjacent_sibling.asp
```

- 多类元素选择器
```html
.p1{color:blue} 
.p2{font-size:30px} 
.p1.p2{font-style:italic} // 这个就是多类选择器
<pp class="p1">hello jesse</p>
<pp class="p2">hello jesse</p>
<pp class="p1 p2">hello jesse</p> // 会具有，blue、30px、italic 这三个样式
```
- 属性选择器
* [title]{} // 选择元素具有 title 属性的元素 
* [href="http://jessechiu.com"]{} // 只匹配属性 href 值为 "Jessechiu.com" 的元素
* [value~="jesse"]{} // 匹配只要属性有 value 且值中有包含 "jesse"，类似 [value*="jesse"]
```html
<pp value="jesse">hello jesse</p> // 匹配
<pp value="hello jesse">hello jesse</p>// 匹配
<pp value="hello je">hello jesse</p>// 不匹配
``` 

#### CSS3 选择器
- 属性选择器
 * [id*=jesse]{} // 包含 jesse 
 * [id^=jesse]{} // 以 jesse 开头
 * [id$=jesse]{} // 以 jesse 为结束字符
 注意：这里是用 id ，因为它也是元素的属性，

- 结构性伪类选择器
* p:first-line{} // 选择第一行元素
* p:first-letter{} // 选择首字母
* li:before/after{ // 元素前面/后面添加内容
		content:"--";
		font-size:30px;
	} 

- 结构元素选择器
* :root{background-color:darker;} // 选择文档中的 root 元素，也就是 html 部分
* div *:not(h1){} // not 排除选择，该列是选择 div 中所有 **子类** 元素，但是不包含 h1 元素
* :empty{} // 用来选择没有内容的元素，通常可以来设置 table 中无值得单元格
* target{} // 用来设置用户所选择的元素样式，例如 a 元素在页内跳转

- li:first-child{} // 选择第一个元素
- li:last-child{} // 选择最后一个元素
- li:nth-child(3){} // 根据当前元素选择下一个元素，可以传人参数：数字，odd:奇数，even:偶数，可以是 an+b 的方式来循环设置样式
- li:nth-last-child(3){} // 上面的相反，是从尾部开始数
注意，如果要以组的方式计算同类元素，应该使用下面的选择器
- h1:nth-of-type(odd){}
```css
	<h1>jesse</h1> // 设置 nth-of-type
	<p>kuku</p>	

	<h1>jesse</h1>
	<p>kuku</p>	

	<h1>jesse</h1> // 设置 nth-of-type
	<p>kuku</p>	

	<h1>jesse</h1>
	<p>kuku</p>
```
- li:only-child{} // 选择只有一个 li 子元素的 li 元素

- 伪类元素选择器(css3 中总共有 17 个)
* input[name="jesse"]:hover/focus/active/checked/invalid/valid{} // 
* input[name="jesse"]:enabled/disabled/required/selection/optional{} // 
* input[name="jesse"]:read-only/default/indeterminate/in-range{} // 

- 通用兄弟选择器
div ~ p{} // 同一父元素之中，某个元素之后，所有其它其它元素，例如
```html
<div> // 相同的父级元素
	<div> // 这个 div 就是指某个元素
		<p>kuku</p>
		<p>kuku</p>
	</div>
	<p>jesse</p> // 选中
	<p>jesse</p> // 选中
	<p>jesse</p> // 选中
</div>
```


### 十二、CSS 动画
- transform: translate(100px,100px); // 表示向 X 坐标、向 Y 坐标，移动
- transform: rotate(45deg); // 旋转角度
- transform: scale(1,2);  // 放大，第一个是 宽度，第二是高度 倍数
- transform: skew(45deg,30deg); // 倾斜角度，向 x 轴，向 y 轴
- transform: rotateX/Y(45deg); // 3D 旋转效果

- 过渡
* transition: 5s;// 过渡时间
```css
div{
	width: 100px;
	height: 100px;
	background-color:blue;
	-webkit-transition:5s; // 这里设置过渡时间
	transition: 5s;
	-weibkit-transition-delay:2s; // 这里可以设置延时 2s 开始,这会导致每次触发都有延时，包含过程状态
	transition-delay:2s;
}
div:hover{// 这里设置触发条件
	width: 200px;
	height: 200px;

	-webkit-transform: rotate(300deg);// 添加过渡效果
	transform: rotate(300deg);
}
```

- 动画
* animation: anim 5s infinite alternate; // anim 是动画名称，对应下面的 @keyframes 动画帧名，infinite 是无限循环，alternate 可以让动画往返动作

```css
.div1{
	width: 100px;
	height: 100px;
	background-color: green;
	-webkit-animation: anim 3s infinite alternate;
	animation: anim 3s infinite alternate;
	position: relative; // 这句是为了可以移动，如果不设置，动不了
}

@keyframes anim{
	// 也可以使用 from、to 来设置过程变化
	0%{background: green;left:0px; top:200px;}
	50%{background: blue;left:200px; top:200px;}
	100%{background: black;left:200px; top:400px;}
}
@-webkiy-keyframes anim{
	0%{background: green;left:0px; top:200px;}
	50%{background: blue;left:200px; top:200px;}
	100%{background: black;left:200px; top:400px;}
}
// ref：http://www.w3school.com.cn/css3/css3_animation.asp
```

### 十三、CSS3 盒子模型
- display:inline/inline-block/block
- display:inline-table // 可以设置 table 以 inline 方式显示
- display:list-item// 可以设置元素按照 li 方式显示，这样就有 li 相对应的样式属性设置
- overflow:hidden/scroll/auto // auto 会自动根据文字内容，在水平和垂直方向添加滚动条
- overfolow-x/y:hidden/scroll/auto // 也可以单独在 x或者y 轴上设置

- box-shadow: 1px 1px 1px red;// 设置盒子阴影，使用方法和 text-shadow 相同。

- 指定盒子大小计算方法
* box-sizing: border-box;// border 和 padding 包含在盒子大小中
* box-sizing: content-box;// border 和 padding 不包含在盒子大小中，显示会大

### 十四、CSS3 背景相关属性
- background-clip: border-box;// 背景色充满 border 和 padding
- background-clip: padding-box;// 背景色充满 padding
- background-clip: content-box;// 背景色 只填充内容部分

- background-origin:border/padding/content-box; // 代表背景从那个区域块开始绘制

- 同一元素绘制多张图片
```html
	div{
		background-image:url(),url();
		background-repeat:repeat-x,no-repeat;// 这里设置第一张图片 x 轴重复，第二张不重复
		background-position: 100%,100%,center,center;// 100% 绘制，居中
	}
	// 显示效果，第一张图片在上面
```
- 使用图片边框，背景图片会根据里面内容自动撑开
- -webkit-border-image:url() 10 20 10 30;
// ref: http://www.w3school.com.cn/cssref/pr_border-image.asp

#### 垂直居中
利用 `top: 50%`(让元素的上边界定位到父元素的50%位置)和`transform: translateY(-50%)`(让元素往上移动元素自身高度的50%).
```css
.vertical-center {
  position: relative;
  top: 50%;
  transform: translateY(-50%);
}
```
